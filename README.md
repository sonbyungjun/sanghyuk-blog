# sanghyuk.dev

손상혁의 개인 블로그

## Tech Stack

- **Framework:** Astro
- **Styling:** Tailwind CSS
- **UI:** React
- **Content:** MDX (Contentlayer)
- **Deploy:** Cloudflare Pages

## Development

```bash
# Install dependencies
yarn install

# Start dev server
yarn dev

# Build
yarn build

# Preview build
yarn preview
```

## Structure

```
├── content/          # Blog posts (MDX)
├── portfolio/        # Portfolio page
├── public/           # Static assets
└── src/
    ├── components/   # UI components
    ├── layouts/      # Page layouts
    └── pages/        # Route pages
```
